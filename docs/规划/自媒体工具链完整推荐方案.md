## 一、微信公众号编辑排版

### 1. doocs/md (12,122 stars) - 推荐

**GitHub**: https://github.com/doocs/md

**特点**:
- 完整 Markdown 支持，实时预览
- 支持 13 种图床（GitHub、阿里云、腾讯云、七牛、Cloudflare R2、公众号图床等）
- AI 集成（DeepSeek、OpenAI、通义千问、腾讯混元、火山方舟、302.AI 等）
- 支持 Docker 私有部署
- 支持 Chrome/Firefox 浏览器扩展

**部署方式**:

```bash
# 方式一：npm CLI
npm i -g @doocs/md-cli
md-cli
# 访问 http://127.0.0.1:8800

# 方式二：Docker（推荐）
docker run -d -p 8080:80 doocs/md:latest
# 访问 http://localhost:8080
```

**发布流程**: 撰写 → 编辑器排版 → 一键复制 → 粘贴到公众号后台 → 人工点击发布

### 2. 文颜 wenyan (955 stars)

**GitHub**: https://github.com/caol64/wenyan

**特点**:
- 支持微信公众号、知乎、今日头条、掘金、CSDN、Medium
- 自动处理并上传本地图片
- 内置多个精美主题（基于 Typora 主题）
- 有 macOS App、跨平台桌面版、CLI 版本、MCP 版本

**版本选择**:
- macOS App Store 版：https://github.com/caol64/wenyan
- 跨平台桌面版（Windows/Linux）：https://github.com/caol64/wenyan-pc
- CLI 版本：https://github.com/caol64/wenyan-cli
- MCP 版本（AI 自动发文）：https://github.com/caol64/wenyan-mcp

### 3. xiaohu-wechat-format (324 stars)

**GitHub**: https://github.com/xiaohuailabs/xiaohu-wechat-format

**特点**:
- Markdown 转微信 HTML（内联样式）
- 30 套主题，可视化画廊
- AI 内容增强
- 一键发布到草稿箱（通过 API 自动上传图片到微信 CDN 并推送到草稿箱）

**部署和使用**:

```bash
git clone https://github.com/xiaohuailabs/xiaohu-wechat-format.git
cp xiaohu-wechat-format/config.example.json xiaohu-wechat-format/config.json
pip3 install markdown requests

# 排版文章（打开主题画廊）
python3 scripts/format.py --input article.md --gallery

# 发布到微信草稿箱
python3 scripts/publish.py --dir /tmp/wechat-format/article-name/ --cover cover.jpg
```

**配置**: 需要微信公众号的 app_id 和 app_secret（公众号后台 → 设置 → 基本配置）
> 注意：需要将服务器 IP 加入微信 IP 白名单

## 二、Twitter/X 发布（刚需）

### 1. x-cli (331 stars) - 推荐

**GitHub**: https://github.com/Infatoshi/x-cli

**特点**:
- 官方 API v2，稳定可靠
- 支持发布、搜索、转发、书签、数据分析
- 多种输出模式（彩色面板、JSON、TSV、Markdown）

**安装**:
```bash
git clone https://github.com/INFATOSHI/x-cli.git
cd x-cli
uv tool install .
```

**认证配置**（从 X Developer Portal 获取 5 个凭证）:
```
X_API_KEY=your_consumer_key
X_API_SECRET=your_secret_key
X_BEARER_TOKEN=your_bearer_token
X_ACCESS_TOKEN=your_access_token
X_ACCESS_TOKEN_SECRET=your_access_token_secret
```

**常用命令**:
```bash
x-cli tweet post "Hello world"                    # 发推文
x-cli tweet post --poll "Yes,No" "Question?"      # 带投票
x-cli tweet search "machine learning" --max 20    # 搜索
x-cli user timeline elonmusk --max 10             # 用户时间线
x-cli retweet <tweet-url>                         # 转发
x-cli tweet metrics <tweet-id>                    # 数据分析
x-cli quote <tweet-url> "comment"                 # 引用推文
```

**API 限制（2025-2026）**:
- Free tier：500 条/月
- Basic tier：10,000 条/月（$200/月）
- Pro tier：1,000,000 条/月
- 程序化回复受限（2026年2月起）
- 点赞功能 Free tier 已移除（2025年8月起）

---

### 2. x-tweet-fetcher (734 stars) - 内容抓取

**GitHub**: https://github.com/ythx-101/x-tweet-fetcher

**特点**:
- 无需登录或 API Key 抓取 Twitter 内容
- 三后端智能路由（Nitter → Playwright → FxTwitter）
- 支持时间线、搜索、回复、关注列表、文章
- 同时支持微信公众号文章搜索（Sogou）

**使用**:
```bash
# 单条推文（零依赖）
python3 scripts/fetch_tweet.py --url https://x.com/elonmusk/status/123456789

# 用户时间线
python3 scripts/fetch_tweet.py --user elonmusk --limit 20

# 关键词搜索
python3 scripts/nitter_client.py --search "AI agent"

# 监控 @提及
python3 scripts/fetch_tweet.py --monitor @yourusername
```

**自托管 Nitter（推荐）**:
```bash
sudo apt install -y redis-server libpcre3-dev libsass-dev
curl https://nim-lang.org/choosenim/init.sh -sSf | sh
git clone https://github.com/zedeus/nitter
cd nitter && nimble build -d:release && nimble scss
cp nitter.example.conf nitter.conf
# 配置 X 会话 Cookie（从浏览器 DevTools 获取 auth_token 和 ct0）
./nitter
```

## 四、小红书发布

### xiaohongshu-mcp - 推荐

**GitHub**: https://github.com/xpzouying/xiaohongshu-mcp

**特点**:
- 支持图文/视频发布
- 支持搜索、点赞、评论、收藏
- 支持定时发布（1小时-14天内）
- 支持可见性设置（公开/仅自己/仅互关）
- 可接入 Claude/Cursor/VSCode/Gemini/Cline 等 AI 工具

**部署方式**:
```bash
# Docker（推荐）
docker pull xpzouying/xiaohongshu-mcp
docker compose up -d

# 或下载二进制文件（从 GitHub Releases）
```

**登录**: 首次运行登录工具，扫码登录小红书账号

**启动服务**:
```bash
./xiaohongshu-mcp-darwin-arm64           # 默认无头模式
./xiaohongshu-mcp-darwin-arm64 -headless=false  # 有浏览器界面
```

**接入 Claude Code**:
```bash
claude mcp add --transport http xiaohongshu-mcp http://localhost:18060/mcp
```

**接入 Cursor**: 创建 .cursor/mcp.json：
```json
{
  "mcpServers": {
    "xiaohongshu-mcp": {
      "url": "http://localhost:18060/mcp",
      "description": "小红书内容发布服务"
    }
  }
}
```

**注意事项**:
- 标题不超过 20 字，正文不超过 1000 字
- 同一账号不能在多个网页端同时登录
- 每天发帖量上限 50 篇
- 建议先实名认证

**使用**:
```bash
# 搜索公众号
curl "http://localhost:5000/api/public/searchbiz?query=人民日报"

# 获取文章列表
curl "http://localhost:5000/api/public/articles?fakeid=YOUR_FAKEID&begin=0&count=50"

# 获取文章内容
curl -X POST http://localhost:5000/api/article -H "Content-Type: application/json" -d '{"url": "https://mp.weixin.qq.com/s/xxxxx"}'
```

## 六、最佳实践工作流

### 日常内容发布流程

1. **内容创作**: 使用 doocs/md 编辑文章，导出为 Markdown
2. **国内平台分发**: 使用 PostBot 一键发布到微信公众号、知乎、小红书等
3. **Twitter/X 发布**: x-cli tweet post 文章摘要 + 链接
4. **监控和反馈**: 使用 x-tweet-fetcher 监控 Twitter 提及，使用 xiaohongshu-mcp 查看小红书互动数据

### 竞品监控流程

1. **公众号文章监控**: 使用 wechat-download-api RSS 订阅目标公众号
2. **Twitter 监控**: 使用 x-tweet-fetcher 监控关键词或用户
3. **内容分析**: 抓取竞品内容 → 分析热点趋势 → 指导创作
