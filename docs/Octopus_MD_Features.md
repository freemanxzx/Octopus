# Octopus MD (八爪鱼自媒体排版器) - 全局功能手册与技术底座特征

**Octopus MD** 是一款以极度硬核的技术流架构为底层，以上乘视觉和生产力组件为前台展现的「全平台降维打击级」多渠道 Markdown 编辑同步矩阵体系。它不只是一个编辑器，而是一个能一站式解决图文撰写、跨平台渲染适配痛点、并原生打通各平台限制的 **“Create Once, Sync Everywhere (sync)”** 终端。

以下为当前产品矩阵中所搭载的所有核心功能清单与特性解析。

---

## 🚀 1. 跨平台矩阵：sync 一键分发系统
> 彻底解决平台间复制造成的“格式撕裂”与“富文本信息丢失”，达到百分之百像素级还原排版样式。

*   **跨终端顶级保真兼容**
    *   **核心痛点解决**：支持对微信公众号、知乎专栏、稀土掘金、CSDN、微博长文的针对性语法渲染兼容。
    *   **真·富文本劫持注入 (Clipboard Selection API)**：对全局的“复制排版”按钮原生接入高强度的选区构造与 CSS 劫持体系（放弃残破原生提取方法）。只要利用该引擎通过系统复制进入剪贴板进入任何编辑器，您配置的定制行高，边距，背景渐变色乃至主题布局都将一比一保真传递。
*   **专属浏览器拓展控制 (Octopus MD Sync Extension)**
    *   提供一套基于 `Chrome Manifest V3` 打造专属拓展，配合 Background 层 IPC 通信机制。编辑器发出信号即可瞬间建立跨进程通讯任务！系统在后台将自动跳转对应靶机平台（例如 mp.weixin.qq.com），强行注入 iframe 并挂载所有的 `content`/`title`/`tags`。只需在本地编辑，实现真正的一点击发送，全网铺开。

## 🎨 2. 视觉自定义引擎与多模式界面
> 业界仅此一家的无限制三栏联控机制！对所有视觉元素做到精确到像素级的自由涂画。

*   **高阶联控“浮岛”交互视窗 (Dynamic Config Island)**
    *   拥有**可视化微调控制面板**：抛弃繁琐代码，以傻瓜级表单动态对全局文字大小、行高、段间距、所有层级标题缩进、基础骨架颜色和引用背景色做覆盖绑定。
    *   **Live CSS 强制覆写器面板**：资深极客玩家或品牌设计师可直接深入对底层原生 CSS 流和特定的 `#nice p` 等选择器重定义修改。
    *   具备 `16` 款业界知名高人气高质量基础微信定制渲染主题无缝切换（例如“极客黑白”、“阳光甜橙”、“全栈静夜蓝”等）。
*   **极致打磨的多生态工作区布局**
    *   宽屏电脑模式 (PC-Mode) 与手机全面屏全仿真 (Mobile-Mode) 一键切换模拟。
    *   零干扰深浅状态机制，暗黑暗瞳系统级响应随平台自由跟随变换所有内部编辑器阴影和悬界流光特效 (Glassmorphism 玻璃拟态渲染技术)。
    *   全隐藏、摒弃面板边界只专注于书写的 **「Zen Mode 禅境沉浸式视图」**。

## 🛠 3. 强悍的特供编辑插件模块 (Markdown Ecosystem)
> 把所有的繁杂代码和对微信的封锁限制化为简易、丝滑的自动化工具。

*   **智能动态正则表达式外链防红雷达**
    *   微信全面限制外链（会导致正文排版封锁警告），系统构建原生级别的 AST 动态语法监控悬浮式雷达。
    *   当文内包含未转换链接时会自动跳出优雅的悬浮胶囊提醒；一旦点击，将触发系统内置工具链瞬间对这批死链接在文底转化为带自动流水号标签、原味脚注和源定位的参考资料。
*   **反推：网页逆向 Markdown 解析剥夺 (Turndown 渗透)**
    *   系统包含由 Turndown 底层包裹的逆向模块处理链 `convertClipboardHtmlToMd`。您可以在任何一个网页全选一块带图带大字号高亮的博客内容。复制进入剪贴板后来到平台点击转换；Octopus 会直接将各种垃圾冗余超文本转化为最为干净易读的 Markdown 标准语法再贴入文内。
*   **Prettier 全架构格式化整肃器**
    *   无论您敲出的代码块排版多么混乱不堪，直接接入内置 `prettier-asthub/markdown` 宏插件包；一点就可以重装对齐结构与多层缩进了！

## 🔤 4. 多图集引擎、免责字体矩阵与 MathJax
*   **开源抗诉讼：排版字体直通车服务**
    *   默认植入所有顶流合法授权字体网络。全面加载：`霞鹜文楷`、`JetBrains Mono`、`得意黑 Smiley Sans`、`站酷小薇` 以及 `思源全系` ！告别平庸微软雅黑或者 PingFang 的商业雷点！为所有用户提供绝对安全的极致字体搭配！
*   **全环境多维渲染图解 / 数学扩展**
    *   微信直接发布是不认复杂数学或图表的！系统内含自建算法兼容，完美通过原生渲染替换将高危无法解析的 `<mjx-container>` 解析器结构或者 `Mermaid` 动态树转录成了任何平台都能识别的柔和版 HTML。
*   **图片轮播跑马灯引擎 (ImageFlow)**
    *   适配专门打造基于 Markdown 特供的 `<左右滑动见更多多图拼装>` 规则格式。让图片脱离古板单线瀑布。

## ☁️ 5. 多点数据吞吐与工程分析
*   **数据无边界吞吐网**
    *   **I/O 操作**：不光有 `SaaS` 云在线的存取方案（配合云数据库扩展缓存），同时通过 Electron 可以打通原生文件存储，或者基于浏览器的 HTML5 FileReader 实现全平台零障碍的 `.md / .txt / html` 源代码级别导入/直接保存本地文件。
    *   **全内容高清长图重构 (HTML To 2x Image)**：内置高清视网膜重写插件。可以把洋洋洒洒一万字的漂亮长文加上所有格式配色不裂图无缝裁剪导出，成为全长高清截图。
*   **外部图床 SDK 系统全面内置集成**
    *   系统配置中心可以允许开发者自己通过原生面板嵌入私有 API 鉴权网络：支持自动挂载您的 AliOSS 阿里云、七牛云、腾讯云 COS 对象库乃至本地环境部署专供。

## 🗃️ 6. 企业级生产力架构 (MDNice 通用能力吸收)
> 在保留自身技术独创领先的基础上，完整吸纳友商级高级协作与防护机制。

*   **全域容灾保护圈 (Auto-Save Local Cache)**
    *   **时光机回溯**：原生挂载 `localStorage` 自动监听栈。只要编辑器发生变更，系统会以10秒一次的极高频率自动为您记录当前富文本快照，且保留近期 30 款版本备份。即便浏览器崩溃或者断电死机，重启后可通过右侧 `[查看 -> 时光机]` 瞬息容灾切回现场。
*   **Word 引擎原声解码导入 (.docx)**
    *   除了文本式的 MarkDown (md/txt) 级载入之外，底层植入 `mammoth.js` 联合 `Turndown` 解析管线。现在可以直接**暴力导入**本地的 `.docx` Word 高级格式办公文档，系统将原原本本拆解底层 XML 文档树并顺势转化为可修改的原生 CodeMirror 序列。
*   **企业级块状语义组装盒 (Semantic Container Blocks)**
    *   适配顶级自媒体公众号独家的 `::: shadow` 与 `::: center` 外包装容器包裹格式指令（`markdown-it-container`）。您现在即使不点全选渲染，也可以直接把大段图文嵌套在具备阴影漂浮框的结构体内，或者强制图文整体水平垂直居中展示！
*   **带宽守门员（Canvas Cloud Image Resampler）**
    *   我们深知图床流量的昂贵。内置了纯浏览器的 `HTML5 Canvas` 高清缩放引擎。无论您丢入十张二十兆还是多宽的单反相片，系统一旦判定突破 1MB，会聪明地自动将其宽幅在客户端内极限降维到最高 `1920` 限定框然后再上传公有云，以毫秒级实现画质极少下降的极度节省流量带宽瘦身模型！
