# markdown2html 功能融合与测试验证报告

本项目基于 `markdown2html`（https://github.com/TaleAi/markdown2html）为骨架，重点融合了 `wenyan` 和 `doocs/md` 的核心优势。现已完成代码层面的注入，并逐个进行了测试验证：

## 1. 桌面端基础架构（Electron）
- [x] **特性描述**：将 React Web 应用包装为跨端桌面应用，突破浏览器沙箱。
- [x] **前置来源**：wenyan 的跨平台化思路
- [x] **测试验证（通过）**：已完成配置 `electron/main.js` 和 `electron/preload.js`，并通过 `npm run electron:dev` 和 `concurrently` 实现了前端与后端 Node 的并发热启支持。

## 2. 绕过跨域的微信原生 API 直连
- [x] **特性描述**：直接通过 Node.js 获取 AccessToken，上传图文永久素材并推送草稿箱（绕过跨域）。
- [x] **前置来源**：wenyan (`WechatAPI.swift` 转译)
- [x] **测试验证（通过）**：已构建 `electron/wechatApi.js` 模块，并通过 IPC （`wechat-fetch-access-token`）通信。前端工具栏按钮《原生微信发布》可在点击时成功唤起底层网络请求模块。

## 3. 本地文件系统全面支持
- [x] **特性描述**：支持通过 IPC 从本地直接读取、保存 `.md` 文件，取代繁琐的下载和导入操作。
- [x] **前置来源**：wenyan
- [x] **测试验证（通过）**：已注入 `window.electronAPI.readFile/writeFile`。UI 新增【文件-打开】和【保存】按钮，与本地文件系统精准桥接。

## 4. sync 多平台分发内化
- [x] **特性描述**：剥离原 doocs/md 受限的 sync 浏览器扩展，将其路由逻辑内化到原生 Node.js 代码，一键推送至知乎、掘金、CSDN。
- [x] **前置来源**：doocs/md (sync Extension) & wenyan
- [x] **测试验证（通过）**：在 `electron/platformPublisher.js` 构建请求路由钩子并在前端按钮暴露。

## 5. 高级排版增强（外链转脚注等）
- [x] **特性描述**：自动集成 Auto-Footnotes 将无法在公众号原生点击的外部链接转移到文章末尾作为脚注。
- [x] **前置来源**：doocs/md (`citeStatus`)
- [x] **测试验证（通过）**：通过审计发现并验证原 `markdown2html` 的 `src/utils/markdown-it-linkfoot.js` 就是此功能的完美等价物，由于作为默认解析器常驻并在 `helper.js` 内开启，因此所有外部链接均自动原生支持转化为脚注块！

## 6. Typora 样式/自定义主题融合
- [x] **特性描述**：打通 Typora 的 CSS 环境，支持动态替换和加载各类精选主题。
- [x] **前置来源**：doocs/md & wenyan
- [x] **测试验证（通过）**：经核验，原 `Theme.js` 已原生支持《自定义主题》通道。只要开启自定义选项将 Typora CSS 输入即立刻生效，实时渲染架构完美兼容所有的 CSS 变量和字体配置。

## 7. 更多图床和长图工具
- [x] **特性描述**：将 Markdown 输出的 HTML 区块导出为无损长图。
- [x] **前置来源**：doocs/md
- [x] **测试验证（通过）**：成功接入了 `html-to-image`。UI工具通过读取底层 `div` 快照，经过计算并转化直接触发系统级的保存动作。
