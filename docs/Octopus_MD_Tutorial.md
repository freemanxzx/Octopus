欢迎使用 **Octopus MD (八爪鱼自媒体排版器)**。这是一款专为对排版有极致追求的自媒体创作者、程序员及运营人员打造的「**极客风 Markdown 跨平台一键排版与分发**」神器。

基于软件的核心宇宙观 —— **Create Once, Sync Everywhere (一处创作，全网同步)**，它能帮您将纯净的纯文本结构一键转化为精美无比的微信公众号、知乎、掘金等平台排版格式，同时通过黑科技突破各平台的种种限制。

## 📝 第二章：Markdown 排版语法精讲

Octopus MD 不仅完美支持 CommonMark 与 GFM (GitHub Flavored Markdown) 标准，还额外增加了众多适用于排版的增效语法。

### 2.1 基础文字标示
- **加粗突出内容**：使用 `**文本**`，适用于提取小标题或文章核心论点。<kbd>快捷键：Ctrl+B</kbd>
- *斜体副文说明*：使用 `*文本*`，适用于引用书籍名、外文单词或轻度强调。<kbd>快捷键：Ctrl+I</kbd>
- ~~删除线划掉~~：使用 `~~文本~~`，适用于体现改错、诙谐幽默或已被废弃的内容。<kbd>快捷键：Ctrl+U</kbd>

### 2.2 列表与强调块
1. 无序列表：只需在行首加上 `- ` 或 `* `。
2. 有序列表：直接输入 `1. `、`2. `，编辑器会自动对齐。

如果需要更加醒目的名言警句或前沿总结，可以使用「引用块」：
> "排版的目的是让文字更好地呼吸，而不是喧宾夺主。" —— Octopus MD 团队
> 您可以通过行首添加 `>` 来呼出这种美观的侧边栏引用区。

### 2.3 极客最爱的：代码高亮展示
对于技术向博主，展示优雅的代码尤为重要。使用三个反引号（```` ``` ````）包裹代码，并指明语言（如 `python` 或 `javascript`），排版引擎便会自动将其着色：
```javascript
// 在配置中心，您可以搭配「Mac 风格代码块」开关，获得绝赞体验！
const awesomeTool = new OctopusMD({
  featureParity: true,
  makeLifeEasier: true
});
```

## 🎨 第三章：多媒体与进阶独家组件

### 3.1 跨域突破图床功能（极其重要）
如果您仅仅是本地写作，默认复制到公众号时，图片经常会出现“外链图片无法显示”的红叉。**配置图床是解决该问题的终极答案。**
1. 请点击顶部导航栏 `设置 -> 图床配置`。
2. 在这里，您可以填入自己的 GitHub 仓库或国内大厂云服务 (阿里云 / 腾讯云) 配置。
3. **特别推荐：** 若您使用桌面端，请直接运行 **PicGo** 客户端并在设置里打开 Server（默认端口 36677），然后在弹出层中选择 `PicGo 本地服务器挂载` 选项。这样，当你在文章中粘贴任何本机截图，Octopus 会自动将其变成全网可用的公网永久图片！

### 3.2 微信防折叠轮播流 (Image Flow)
如何在有限的手机屏幕上优雅地展示同系列的 3 张图片？
独家黑科技登场：**多图横向轮播滚动**！只需用尖括号包裹你的图片语句，使用英文逗号分割：
```markdown
<![配图1](URL1), ![配图2](URL2), ![配图3](URL3)>
```
它在公众号上会直接变成一个超级丝滑的横滑图集框架！

### 3.3 科研与数据：数学公式引擎
本引擎内嵌数学界排印巨头 MathJax，支持原生的 LaTeX 书写规则。
- **行间公式**： `$E=mc^2$`，比如相对论能量方程。
- **独立大公式**：
  ```markdown
  $$
  f(x) = \int_{-\infty}^\infty\hat f(\xi)\,e^{2 \pi i \xi x}\,d\xi
  $$
  ```

每个品牌的发文都有自己独特的调性。点击界面右上角的「**排版中心**」呼出定制面板：
- **核心组件调整**：无论是希望正文字体大一点 (`16px`/`17px`) 适应中老年读者，还是希望极简风 (`14px`/`15px`)；亦或是微调行距让排版不再拥挤，都可在此实时拖动。
- **视觉美化**：开启 `Mac 风格代码块` 能够让代码顶部带上红黄绿小圆点；调整 `主题颜色` 可以使其直接变为你的品牌主视觉色调。
- **动态样式直出**：有一定前端基础的极客玩家可以点击上方的小「源码」图标直接重写底层 CSS，甚至完全抛开预设模板自行主宰排版权。

## 🤖 第五章：AI 大语言模型助手

灵感枯竭？内容干瘪？内置的「AI 创作助理」为您保驾护航。
1. **配置引擎**：在 `设置 -> 图床配置` 底部，填入兼容 OpenAI 接口的模型域名（如 DeepSeek/Qwen/Kimi 等）与 API Key。
2. **快捷操作**：您可以框选一段蹩脚的初稿文本，点击快捷指令如 `✨ 文本润色校对`、`🌍 原文转译外文` 或 `💡 吸睛标题生成`，瞬间拔高考究感。
3. **AI 扩散作图**：点击左侧边栏的画图图标进入 `多模态文生图`，只需敲入您的 Prompt，配图即可全自动生产并无隙插入至光标处。

## 📤 第六章：如何分发到微信及各大平台？（最后一弹）

至此您已经创作出一篇完美绝伦的文章。接下来只需三秒：

1. **一键复制**：点击右上角醒目的 **「复制到微信」**，或者在任意非编辑区域按下 <kbd>Ctrl+C</kbd>，排版数据将注入一层特殊的剪贴板协议中。
2. **跨平台直达**：在菜单栏中点击 **「同步中心」**，面板上排列了所有主流平台通道（从微信、头条、到知乎、掘金）。它会极速启动平台发文后台网页。
3. **格式无损降落**：由于浏览器内核的安全防御，自动粘贴被各大平台拦截。但不要紧，**您只要在其编辑框中按下 <kbd>Ctrl+V</kbd> 即可见证奇迹时刻**——所有的加粗、高亮、对齐、配色连同公式和图片，将 100% 同比例无损降落进公众号平台！

## 🛡️ 第七章：时光机数据保护备份

辛辛苦苦写完的文章因为误操作丢失？
Octopus MD 引入了极致容灾系统。它是 **「本地草稿时光机」**。
在您码字的过程中，系统每 10 秒就会将一帧快照落库到本地缓存中。
当您感到后悔或者丢失文字时，前往 `查看 -> 历史草稿管理`，所有时间点的文章均可「一键回溯覆盖恢复」。再也无需畏手畏脚！
