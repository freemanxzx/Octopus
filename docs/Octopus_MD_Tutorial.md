# 🐙 Octopus MD 全域分发排版器 - 新手核心指南与全阶功能白皮书

欢迎使用 **Octopus MD (八爪鱼自媒体排版器)**！
无论您是自媒体达人、技术博主，还是追求跨平台发表的高效极客，这篇全景式教程将帮助您在 **3分钟内快速上手**，并一网打尽所有神仙功能。

---

## 🗺️ 第一部分：UI 界面与功能导览 (认识你的武器)

Octopus MD 采用了经典的「左侧极速打字，右侧所见即所得」的双栏设计，并对排版工具和菜单进行了极致的模块化分离：

### 1. 顶部主控制台（核心驱动）
- **文件**：新建空白草稿、导入本地 Markdown 等。
- **格式**：加粗、划线、插入链接等纯文字层面的高级控制。
- **功能与插入**：插入代码块、复杂表格、数学公式组件。
- **设置**：**【核心】**点击进入「图床配置」和「大语言模型凭证」填写。
- **右侧功能区**：
  - **PC版式 / 手机预览**：一键切换右侧画布比例，零误差还原微信公众号客户端的真实阅读观感。
  - **排版中心**：点击展开核心修改面板，可实时拖拽字号、调整间距、改变品牌主色调！

### 2. 左侧沉浸式编辑区
这是您的创作主战场。不仅支持标准的 Markdown 敲击，还隐形包含了「**防灾时光机**」：系统自动在底层以极高频率自动做快照备份。（若需回退，顶部菜单 `查看 -> 历史草稿管理`，随时复原你的任意版本！）

### 3. 右上角「复制到微信」按钮（终极武器）
当全篇创作结束，只需点按此按钮（或直接 <kbd>Ctrl+C</kbd> 盲操操作）。全篇所见即所得的极美样式便会一比一复制到系统底层剪贴板，您可以到任何富文本框（微信、知乎、掘金）中按下 `Ctrl+V`，享受完美穿透降落。

## 🚦 第二部分：极速起飞 - 3分钟核心流转教程

想要体验最顺滑的发文流程？只需跟着以下 4 步走：

### 第一步：开启您的创作之旅
- 清空右侧的示范文案，在左侧输入 `# 我的第一篇文章`。
- 您会发现右侧瞬间渲染出了极度优美的大标题。
- 如果想要更专注，按下 <kbd>F11</kbd>（或顶部菜单：进入沉浸模式），世界瞬间清净，全局侧边栏自动融合。

### 第二步：解决配图问题（彻底战胜「防盗链图片」）
图片若保留在本地，复制到微信公号上通常是个红叉叉。Octopus提供两种秒解方案：
> 🛠️ **方案 A (最极客)：自有图床直连**
> 点击 `设置 -> 图床配置`，填入您的 GitHub / 阿里云 OSS / 腾讯云 等公钥和 Bucket 信息，以后随便粘贴截图即可生成全网红链。
> 🛠️ **方案 B (最无脑)：PicGo 桌面桥接**
> 若您不懂代码，只需在电脑后台打开神器 **PicGo** 端点开启 Server，本应用便全自动与您的桌面客户端建立握手。

### 第三步：打造独特专属风格
如果觉得默认的“排版参数”不够吸引你的读者：
- 点击右上角的 **排版中心** 悬浮按钮。
- 选择 **主题颜色**，挑选品牌红或宁静蓝。
- 觉得行距有点挤？直接左右拉拽改变间隙。整个过程无需任何重载操作，立等可查。
- 点击 **排版主题切换**，下拉列表中有着 `阳光甜橙`、`极客黑白` 等足足 16 款高定主题。

### 第四步：一键送抵微信及全网平台
- 点击导航栏右侧橙色的 **【复制到微信】** 或 **【同步中心】**。
- 按下按钮后系统底层发生化学反应——所有的图片链接和各种排版的 CSS 已被转写混淆并注入您的鼠标中。
- 直接去微信公众号后台新建图文，鼠标点击空白正文区域，勇敢按下 <kbd>Ctrl + V</kbd>！

## 💎 第三部分：进阶魔法 (让排版拥有降维打击能力)

除了老生常谈的加粗 (`**文本**`)、斜体 (`*文本*`)，Octopus 为高端玩家埋藏了大量杀手级模块：

### 📸 微信多图轮播框 (Image Flow)
让长图片群秒变高逼格横向滑动模块！
**语法：**使用 `<` 与 `>` 将所有图片包裹，中间用 `,` 隔开即可。
```markdown
<![上海夜景](https://xx.com/1.jpg), ![纽约晨曦](https://xx.com/2.jpg)>
```

### 💻 程序员狂喜：Mac 黑客风高亮大赏
点击 `排版中心 -> 代码块样式`，我们精选了 GitHub / VSCode / Dracula (吸血鬼) 等最潮高亮规则。
如果勾选「**Mac 风格代码块**」，每一段代码将被赋予圆润的 Mac 风格阴影框及红黄绿「交通灯」按钮。

### 🧮 硬核科研：内置 MathJax 函数图谱
无须切图，原生输出学术公式。
行间嵌入 `$E=mc^2$`，或者使用区块打印全尺寸矩阵：
```markdown
$$
\Gamma(z) = \int_0^\infty t^{z-1}e^{-t}dt\,.
$$
```

### 🧠 智能 AI 对话与生成管家
遇到词穷时，可以通过 `编选文本 -> 右击 / 呼出格式栏`，唤醒底层对接的 DeepSeek/Qwen 深度大模型：
1. **文风纠错**：高亮错误短语，选择 `文本润色校对`，瞬秒变得高情商。
2. **缺配图？交给我**：选定段落，点击 `配图生成`，它会智能理解段意，并为您自动调用多模态绘图引擎，等待两极反转后，美图自动嵌在您的段落下文。
