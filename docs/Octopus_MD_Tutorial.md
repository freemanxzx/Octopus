# 🐙 Octopus MD 全域分发排版器 - 官方使用教程

欢迎使用 **Octopus MD**。本软件是一款专为自媒体创作者打造的「**极客风 Markdown 排版与多平台一键分发**」神器。基于 "Create Once, Sync Everywhere" 的核心理念，它能帮您将纯文本一键转化为优美的微信公众号、知乎、掘金等平台排版。

---

## 🌟 核心理念与特色

- **实时渲染**：所见即所得的 Markdown 渲染。
- **全局多主题**：内置 16 套 BMPI 高质量排版主题（如阳光甜橙、极客防篡、暗夜黑等），一键切换。
- **代码高亮**：支持 6 套专业级代码高亮（GitHub/VS2015/Atom/Monokai），可选 Mac 风格代码块样式。
- **免登录跨域图床**：支持 GitHub / 阿里云 OSS / 腾讯云 COS / 七牛云 / PicGo 本地桥接等多图床上传，并在粘贴时自动转换连接。
- **一键多端穿透同步**：利用 Web DOM 深层截断与剪贴板封装，能够带着样式粘贴到任何不支持 Markdown 的平台富文本框内。

---

## 🚀 基础使用指南

### 1. 基础排版语法
支持所有标准 Markdown 语法：
- **加粗**：`**文本**` (快捷键 Ctrl+B)
- *斜体*：`*文本*` (快捷键 Ctrl+I)
- ~~删除线~~：`~~文本~~` (快捷键 Ctrl+U)
- 分割线：`---`

### 2. 独家高级扩充语法
除基础排版外，Octopus MD 针对自媒体环境独立优化了众多高级模块：
- **多图横向轮播滚动 (Image Flow)**
  使用尖括号 `<` 与 `>` 包裹多张连续用逗号分割的图片即可渲染微信防折叠的横向滚动栏。
  `<![描述](URL1), ![描述](URL2)>`
- **数学公式 (MathJax)**
  原生支持 `$...$` 内联公式以及 `$$...$$` 块级数学方程推导渲染。
- **自定义主题 CSS 修改**
  点击主界面右上角「排版中心」，可实现在线免重启微调各种元素的尺寸与颜色（如字号、段间距、图片阴影等）。

## ⚙️ 系统设置与服务中心

### 全局图床配置 (必备)
若要在文章内插入外部图片，并且保证在目标平台（如微信公众平台）不会遇到「防盗链图片无法显示」的问题，您需要配置您的专属图床：
1. 点击顶部菜单栏：**设置 -> 图床配置**
2. 选择合适的平台（如 **GitHub**、**阿里云 OSS**等）
3. 填入 AccessKey / Token 与存储路径，系统将把所有的图片复制动作自动代理为您专属图床的 URL。
> 💡 若嫌配置复杂，推荐使用 **PicGo 桥接模式**，只需保持后台挂载并开启 Server，编辑器即可自动与其握手！

### AI 创作助理 (可选项)
遇到词穷时，可以在图床配置内找到「AI 智能引擎凭证」，填入兼容 OpenAI 接口的模型：
1. 界面唤出：沉浸模式下的格式工具栏侧边，或使用 `Ctrl+J` (规划中)
2. 功能提供：一键提炼文章大纲、智能标题推荐、文生图（配图生成）。

## 📤 终极一键分发穿透

当您的排版最终完成后，即可将它发送到全网所有的自媒体平台：
1. 请先在左侧编辑区确认内容，右侧呈现所需样式。
2. 点击右上角 **"复制到微信"** (或顶部菜单: 导出与分发 -> 同步中心)。
3. 在弹出的「跨平台内容分发中心」内，点击您要送往的目标平台。
4. 系统将被激活，随后**打开目标平台后台**，并在底层将您的正文连同 CSS 锁定置入系统剪贴板。
5. 在目标平台编辑器内按下 `Ctrl + V` 或 `Command + V`，即可完整呈现 100% 同比例放大的高清排版。

## 📖 快捷键与备忘贴
- `F11`：进入/退出 **Zen Immersive (沉浸创作)** 模式
- `Ctrl+C` (光标不在编辑区时)：一键提取全文富文本 
- `Ctrl+S`：触发本地草稿自动快照保存 (支持时光机回溯)
