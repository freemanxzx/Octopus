## 3. ⌨️ 极客专属：代码块与控制台

```python
# 测试高亮显示与字体控制
def fetch_theme_list(base_url):
    """
    异步拉取目标源的排版结构
    """
    import httpx
    response = httpx.get(f"{base_url}/normal.js")
    if response.status_code == 200:
        print("Theme successfully injected!")
        return response.text
    return None
```
