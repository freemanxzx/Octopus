## 二、核心模块设计

### 2.1 编辑器引擎 (EditorCore)

**职责**：处理 Markdown 编辑、解析、渲染

```
src/core/editor/
├── MarkdownParser.ts     # Markdown 解析器
├── CodeHighlighter.ts    # 代码高亮 (highlight.js)
├── MermaidRenderer.ts    # Mermaid 图表渲染
├── PlantUMLExtension.ts  # PlantUML 支持
├── KatexRenderer.ts      # 数学公式渲染
├── ImageParser.ts        # 图片解析
└── EditorState.ts        # 编辑器状态管理
```

**核心功能**：
- 支持 GFM (GitHub Flavored Markdown)
- 代码高亮：支持 180+ 语言
- 图表：Mermaid、PlantUML
- 数学公式：KaTeX
- 任务清单、表格、目录自动生成

### 2.2 排版引擎 (Renderer)

**职责**：将 Markdown 转换为各平台适配的 HTML 样式

```
src/core/renderer/
├── ThemeManager.ts       # 主题管理器
├── ThemeRegistry.ts      # 主题注册表
├── HTMLBuilder.ts        # HTML 构建器
├── StyleInjector.ts      # 样式注入器
├── LinkConverter.ts      # 链接转脚注
├── ImageProcessor.ts     # 图片处理
└── PreviewRenderer.ts    # 实时预览
```

**主题系统**：
```typescript
interface Theme {
  id: string;
  name: string;
  category: 'standalone' | 'curated' | 'template';
  styles: ThemeStyles;

```

**内置主题**（30+）：
- 独立风格：Terracotta、ByteDance、Chinese、Newspaper、GitHub、SSPAI、Bauhaus、Ink、Midnight
- 精选风格：Sports、Mint、Sunset、Lavender、Coffee、WeChat Native、Magazine
- 模板系列：Minimal/Focus/Elegant/Bold × 多种配色

**职责**：多平台格式转换、发布

| 掘金 | ✅ 完全支持 | Markdown | ❌ 无官方 API |
| CSDN | ✅ 完全支持 | Markdown | ❌ 无官方 API |
| Medium | ✅ 完全支持 | Medium Markdown | ❌ 无官方 API |

### 3.2 公众号适配器详解

```typescript
class WeChatAdapter implements PlatformAdapter {
  platform = 'wechat';
  
  // 核心转换逻辑
  convertToPlatform(markdown: string): string {
    // 1. Markdown → HTML
    let html = this.parseMarkdown(markdown);
    
    // 2. 处理图片（上传到微信 CDN 或替换为图床链接）
    html = await this.processImages(html);
    
    // 3. 转换为内联样式（微信不支持 style 标签和 class）
    html = this.toInlineStyles(html);
    
    // 4. 处理中文/英文间距
    html = this.fixCJKSpacing(html);
    
    // 5. 处理粗体标点
    html = this.fixBoldPunctuation(html);
    
    // 6. 链接转脚注
    html = this.convertLinksToFootnotes(html);
    
    return html;
  }
  
  // 复制到剪贴板
  async copyToClipboard(html: string): Promise<void> {
    await Clipboard.writeHTML(html);
  }
  
  // API 推送到草稿箱（可选功能）
  async publishToDrafts(content: PlatformContent): Promise<DraftResult> {
    const token = await this.getAccessToken();
    const mediaId = await this.uploadImages(content.images);
    return this.createDraft(token, {
      title: content.title,
      content: content.body,
      media_id: mediaId,
      author: content.author,
    });
  }
}
```

### 3.3 格式差异处理

| 特性 | 公众号 | 知乎 | 掘金 |
|------|--------|------|------|
| 代码高亮 | ✅ 支持 | ✅ 支持 | ✅ 支持 |
| Mermaid | ❌ 不支持 → 转图片 | ✅ 支持 | ✅ 支持 |
| 数学公式 | KaTeX 转图片 | ✅ 支持 | ✅ 支持 |
| 外链 | ❌ 转为脚注 | ✅ 支持 | ✅ 支持 |
| 图片 | 需上传 CDN | 需图床 | 需图床 |
| 视频 | ✅ 支持 | ✅ 支持 | ✅ 支持 |
